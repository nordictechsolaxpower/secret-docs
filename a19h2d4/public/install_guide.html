Installationsguide

Version 1.0 · Uppdaterad: 2025‑11‑06

Syfte

Denna guide beskriver hur du publicerar dokument på en hemlig GitHub Pages‑sajt, som inte indexeras av sökmotorer.

Struktur

/ (root)
├─ .nojekyll
├─ robots.txt
├─ index.html
└─ public/
   ├─ install_guide.html
   └─ faq.pdf

Publicering

  1. Lägg till filer via GitHub web (Add file → Upload files).
  2. Commit → avvakta 30–60 sek för att Pages ska bygga.
  3. Testa via din publicerade URL.

Säkerhet & sekretess

  • Noindex: Varje sida har <meta name="robots" content="noindex, nofollow">.
  • robots.txt: Disallow: / blockerar indexering.
  • Obskyr URL: Använd en svårgissad path, ex. /a19h2d4/.
